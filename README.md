PosterTrack — Sistema de Gestión de Inventario

Sistema web para la gestión profesional de posters de películas y series.

Demo en vivo
Ver sistema desplegado

https://kingalandog.github.io/inventrackk


PosterTrack permite gestionar el inventario de posters de películas y series,
con control de stock, alertas y reportes visuales.

Funcionalidades 

- Dashboard con estadísticas en tiempo real
- Catálogo con filtros por género
- Sistema de alertas de stock bajo
- Reportes gráficos por género
- Agregar y eliminar posters

Herramientas
- HTML5
- CSS3
- JavaScript
